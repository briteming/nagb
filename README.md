## 博客

### 关于这个项目

如你所见，一个博客...

利用 GitHub Issues 进行 post 的后台管理，基于 Next + antd

### 部署

1. 在 `src/constants/ApiUrlForBE.js` 修改你需要关联的 GitHub 仓库的地址
2. 部署在 [now](https://now.sh)，注册账号，关联 GitHub，`build` 即可

### 自定义

`src/constants/ConstTypes.js` 修改各种属性
